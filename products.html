Система нарощування нігтів гелем Brisa від CND

Гелева система Brisa для моделювання (нарощування) нігтів ідеально підходить для роботи на формах, типсах або на натуральних нігтях. Стійка до пожовтіння формула гарантує збереження початкового кольору. Засновані на запатентованій технології ковалентного зв'язку та на посиленій взаємопроникній структурі, гелі Brisa від CND створюють міцне та носибельне нарощування. Утворюють міцний зв'язок із кератином нігтя для максимальної міцності.

ПОЗИЦІОНУВАННЯ:

1. Чому CND PLEXIGEL™?

  • CND PLEXIGEL™ являє собою універсальну гелеву систему у флаконі для моделювання та зміцнення натуральних нігтів, яка надає міцності, довжини та форми на більш ніж 3+ тижні.
  • Без ємностей.
  • Без додаткових аксесуарів.
  • Не вимагає змішування.
  • Жодних зайвих кроків.
  • Може використовуватися з усіма видами кольорового покриття від CND™, включаючи гель-лак, Shellac та Vinylux для створення бездоганних нігтів!

2. Який рівень кваліфікації потрібний для роботи з CND PLEXIGEL™? CND PLEXIGEL™ розроблений для будь-якого професійного рівня.

  • ПОЧАТКОВИЙ: Shaper - це проста альтернатива зміцнення натуральних нігтів, що наноситься пензлем.
  • СЕРЕДНИЙ РІВЕНЬ: Забезпечує швидкість та зручність за рахунок скорочення вдвічі часу на корекцію та обпилювання.
  • ПРОФЕСІЙНИЙ: Розширює спектр послуг салону, забезпечуючи повний творчий потенціал з кришталевою чіткістю.

3. Чим CND PLEXIGEL™ відрізняється від інших типів нарощування?

  • CND™ надає додаткові можливості у моделюванні нігтів, тому спеціаліст з манікюру може адаптувати послуги відповідно до потреб клієнта в залежності від типу нігтів та способу життя.
  • CND PLEXIGEL™ пропонує зручний та універсальний спосіб надати нігтям форми та довжини.
  • CND PLEXIGEL™ видалений.

ФОРМУЛА:

  • Чи є CND PLEXIGEL™ розчинним або видаленим? CND PLEXIGEL™ вимагає спилювання. Час видалення становить приблизно 20-30 хвилин і може змінюватись в залежності від товщини продукту. Для ефективності ми рекомендуємо зменшити товщину на 95% за допомогою фрезера перед обгортанням нігтя зволожуючим засобом для зняття CND™ OFFLY FAST™ або ацетоном та подвійним ватним диском.
  • CND PLEXIGEL™ формулою без 12-ти шкідливих CND PLEXIGEL™ був розроблений без наступних 12 інгредієнтів: Формальдегід, толуол, DBP(дибутилфталат), формальдегідна смола, камфора, ксил , парабени, етилтозиламід, трифенілфосфат TPP), MIT, інгредієнти тваринного походження та глютена.
  • Чи є формула CND PLEXIGEL™ веганською? CND PLEXIGEL™ це веганський продукт, створений без інгредієнтів тваринного походження.
  • Чи проводилися досліди над тваринами при створенні формули CND PLEXIGEL™? CND PLEXIGEL™ не тестувався на тваринах.
  • Чи містить CND PLEXIGEL™ мономери HEMA(Гідроксиетил-метакрилату)? CND PLEXIGEL™ розроблявся без використання мономерів типу HEMA.
  • Чи є CND PLEXIGEL™ системою без запаху? Так, хоча ви можете помітити легкий запах розчинників, що містяться в ґрунтовці Bonder і закріплювачі Protector Top Coat.
  • Чи має CND PLEXIGEL™ схожу формулу з пролонгованим базовим покриттям CND™ SHELLAC™ Wear Extender Base Coat? Ні. Пролонговане базове покриття CND™ SHELLAC™ Wear Extender Base Coat розроблене для зчеплення з гель-лаком CND™ SHELLAC™. Грунтовка CND PLEXIGEL™ Bonder призначена для роботи тільки з Shaper та Builder.
  • У чому різниця між CND PLEXIGEL™ Shaper та Builder ? CND PLEXIGEL™ Shaper - це кристально-прозорий гель, що наноситься пензлем, який надає нігтям об'єму, вирівнює їх і надає форми. Ідеально підходить для зміцнення натуральних нігтів та створення невеликої довжини. Його кристально-прозора формула відмінно підходить для нейл-арту! CND PLEXIGEL™ Builder трохи щільніший за в'язкістю і, отже, легше створювати довше нарощування, що потребує високого апексу та архітектури.

ЗАСТОСУВАННЯ / ВИКОРИСТАННЯ:

  • Чи слід CND PLEXIGEL™ струшувати перед використанням? Shaper та Builder струшувати не обов'язково, проте для Bonder та Protector Top Coat ми рекомендуємо акуратно обертати пляшку, щоб змішати інгредієнти, оскільки вони насправді містять розчинники.
  • Чи потрібно зашліфувати натуральний ніготь перед нанесенням CND PLEXIGEL™? CND PLEXIGEL™ є системою для гелевого нарощування, яка вимагає P.R.E.P для належної адгезії. Необхідне легке видалення блиску за допомогою пилки абразивністю 240 або насадки для фрезера середньої зернистості.
  • P- Виконати манікюр
    R- Зняти глянець
    E- Очистити поверхню та видалити забруднення.
    P- Знежирити нігтьову пластину.
  • Чи можна носити CND PLEXIGEL™ окремо без кольору? Так. При використанні як самостійної системи CND PLEXIGEL™ можна носити окремо для створення прозорих нігтів. Завершіть моделювання за допомогою закріплювача CND PLEXIGEL™ Protector Top Coat терміном від 3 тижнів.
  • При нанесенні кольору гель-лаку CND™ SHELLAC™ поверх CND PLEXIGEL™ слід видаляти липкий шар (клейковину)? Якщо змодельовані нігті не вимагають надання форми або об'єму, ви можете нанести гель-лак CND™ SHELLAC™ безпосередньо поверх липкого шару.
  • Чи потрібні два шариCND PLEXIGEL™ Shaper ? Так, ми рекомендуємо два шари для кращого носіння. Нанесіть перший шар тонким шаром (згідно з протоколом) для оптимального контролю комфорту та кращої адгезії, а другий шар – у міру необхідності для кожного клієнта.
  • Чи можна використовувати закріплювач CND™ SHELLAC™ поверх CND PLEXIGEL™ замість верхнього покриття Protector Top Coat ? Для отримання прозорого моделювання ми рекомендуємо нанести верхній шар CND PLEXIGEL™ Protector Top Coat . Якщо ви використовуєте гель-лак CND™ SHELLAC™ поверх CND PLEXIGEL™ , завершіть роботу закріплювачем CND™ SHELLAC™ Original (або DURAFORCE ™ ) замість верхнього покриття Protector Top Coat.
  • Чи можна використовувати матеріали для нейл-арту (наприклад, пігмент, блискітки, прикраси тощо) з CND PLEXIGEL™ ? CND PLEXIGEL™ - ідеальна основа мистецтва. Ми рекомендуємо нанести базовий моделюючий шар системи CND PLEXIGEL™ (Bonder, Shaper / Builder та Protector Top Coat) перед нанесенням художніх елементів. Це захистить цілісність прикраси під час носіння. Щоб потім додати елементи мистецтва за бажанням, у тому числі вбудовування та обволікання прикрас або змішування пігментів та добавок у CND PLEXIGEL™ на верхньому шарі.
  • Чи можна наносити CND PLEXIGEL™ дуже тонким шаром? Так. Якщо у вашого клієнта з'являються тріщини або відколи, можливо, ви нанесли занадто тонкий гель. Спробуйте нанести додатковий шар і переконайтеся, що краї всіх шарів запечатані для кращого зношування.
  • Чи можна використовувати базове покриття CND™ SHELLAC™ замість CND PLEXIGEL™ Bonder ? Ні. Щоб гарантувати довговічність послуги CND PLEXIGEL™ , необхідно використовувати CND PLEXIGEL™ Bonder, щоб міцно зчепити змодельовані нігті, і він діє як сполучний шар під час роботи.
  • Чи можна просушувати CND PLEXIGEL™ в УФ-лампі від CND™ ? УФ-лампа від CND™ має широкий розподіл довжини хвилі і має добре працювати, однак, оскільки CND™ більше не виробляє УФ-лампи, ми не тестували її з CND PLEXIGEL™ . Ми рекомендуємо світлодіодну LED лампу CND™ як найкращий вибір для належного затвердіння. CND PLEXIGEL™ відкалібрований для належного затвердіння в обох наших світлодіодних лампах CND™ (закритих та відкритих). Використання іншої лампи може призвести до недостатнього або надмірного затвердіння і може вплинути на результати зношування продукту.
  • Чи можна затвердити CND PLEXIGEL™ за допомогою переносної мініатюрної світлодіодної лампи? Мобільні або міні-світлодіодні ліхтарі мають широкий діапазон світловіддачі, що ускладнює калібрування їх ефективності при правильному затвердінні CND PLEXIGEL™ . Ми рекомендуємо використовувати світлодіодну лампу CND™ , яка відкалібрована для роботи в якості системи разом із протоколом CND™ Custom Cure , щоб забезпечити належне та комфортне моделювання.
  • Чому деякі клієнти відчувають печіння під час затвердіння? CND PLEXIGEL™ - це формула, багата на мономери і, деякі клієнти можуть відчувати теплову чутливість. Протокол CND ™ Custom Cure .
  • Чим корисний режим спалаху (блимання) у лампах CND™ LED Lamp ? Миттєве затвердіння означає часткове затвердіння гелю за 2-3 «спалахи» у світлодіодній лампі CND™ для зручності клієнтів та контролю якості продукту. Використання світлодіодної лампи CND™ з кнопкою 1 протягом 2-3 імпульсів зменшить потужність на 50%, що дозволить контролювати теплові відчуття. Після того, як гель нанесений на всю руку, затвердіть протягом однієї хвилини за допомогою кнопки попереднього налаштування 2B для повного затвердіння.

УПАКОВКА / ЦІНА:

  • Для чого призначена велика шийка флакона? Місткість CND PLEXIGEL™ була розроблена спеціально для простоти використання з урахуванням індивідуальної в'язкості кожного продукту. Шийка флакона більшого розміру враховує в'язкість гелю, що полегшує його вилучення з ємності та контроль порції при кожному нанесенні.

Шелак (Shellac) - гель-лак для нігтів від CND

Shellac (Шелак) - це революція у світі краси. Ідеальний яскравий та барвистий манікюр, який дозволить насолоджуватися красою рук цілих 2 тижні. Шелак - це справжня мрія: ідеальний стійкий манікюр та захист для ваших нігтів. Покриття Shellac засноване на запатентованій UV3 технології, яка об'єднала у собі простоту нанесення лаку та стійкість гелю до сколів та подряпин.

У Шеллаку поєднані найкращі Показники гелю - тривала шкарпетка і стійкість до ушкоджень. З іншого боку, у ньому присутні найкращі якості професійного лаку - різноманітна за кольорами та кількістю палітра відтінків та простота використання. Шелак для нігтів дуже зручно і просто наноситься, висихає протягом двох хвилин в УФ-лампі, за рахунок чого виключається можливість змащування. При необхідності він легко видаляється без спилювання, спеціально створеною рідиною для зняття гель-лаків, тому ідеально підходить для манікюру та педикюру.

Покрокову інструкцію щодо нанесення Шеллака можна переглянути тут.
